# knockbit_neopixel

Control micro:bit by wechat app with neopixel.

## Feature

- Designed for wechat(Mini Program) knockbit over microbit
- Show Image with yourself.
- Play music by key.
- Transmit messages between microbit and knockbit.
- With neopixel driver.
- Do more thing with the message block.enjoy!

## Log

2019-02-12 此版本是最小版本的蓝牙控制包含了neopixel led控制(neo)
1、只保留了microbit基本功能和neopixel
2、增加了蓝牙连接时，蓝牙断开时的额外命令，可以用来初始化/清理状态（初始化可以放入set命令中）
3、增加了读取传感器的功能，同样在小程序中增加了对应模块
4、修复了按键音乐播放的一个小bug
